WAMP v1 Demos: Spring 4 / springwamp / AutobahnJS

Demos

  • Ext JS 4 Grid    SockJS    Native WebSocket

    Demonstrates an Ext JS store and model with a WAMP proxy. Create, update and delete operations are broadcasted to all connected clients.

  • Chat    SockJS    Native WebSocket

    This is the sample application from portal-java.

  • Smoothie Charts    SockJS    Native WebSocket

    Copy of the Smoothie Charts Server Load Example. In this example the server generates the random data every second and sends it to the connected clients.

  • Google Maps    SockJS    Native WebSocket

    Server sends periodically coordinates to the connected clients that moves two car icons on a google maps.

  • Snake    SockJS    Native WebSocket

    Port of the WebSocket snake demo included in the Tomcat (7 and 8) distribution.

  • Realtime Bandwidth Monitoring    SockJS    Native WebSocket

    Shows received and sent bytes of this server network interface. Port of the pesarkhobeee/Realtime-Bandwidth-Grapher project.
